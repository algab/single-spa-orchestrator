# Single-spa Orchestrator

Single-spa orchestrator creation without using any module bundlers.

## User Guide

1 - Run mfe page-one

```
cd page-one
yarn start
```

2 - Run mfe page-two

```
cd page-two
yarn start
```

3 - Run mfe header

```
cd header
yarn start
```

4 - Run index.html file on a web server

```
npx http-server -p 4000 .
```
